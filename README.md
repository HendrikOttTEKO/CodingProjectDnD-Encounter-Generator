# CodingProjectDnD-Encounter-Generator

Um das Programm debuggen zu können ist die Live Server extension in VS Code zwingend. Der Port im launch.json file ist auf 5500 zu setzen.
Es können maximal 20 PDFs pro Tag zu json Files umgewandelt werden.
Eine Test Funktion zur direkten injektion von den Json Daten ist gegeben.
Die Serverside Logik ist in dem txt File zu entnehmen. Aus Sicherheitsgründen ist kein API Key vorhanden. Der Link in dem Programm funktioneiert einwandfrei
